Habilidades

Java

Es un lenguaje de programación orientado a objetos, cuya implementación va desde juegos hasta aplicaciones móviles, incluyendo software web y empresarial. Mis habilidades en este lenguaje estan orientadas principalmente al desarrollo backend y software empresarial.

C#

Es un lenguaje de alto nivel orientado a objetos,utilizado para desarrollo de servicios en la nube, aplicaciones web y servicios backend. Mis habilidades en este lenguaje consisten en el desarrollo de software de escritorio y videojuegos.

HTML

Es un lenguaje utilizado para la elaboración de páginas web. También permite el navegar entre páginas, interactuar con un usuario y desplegar información de forma eficaz, organizada y compatible. Mis habilidades en este lenguaje consisten en el desarrollo de diversos sitios web con y sin consumo de API's.

CSS3

Es un lenguaje de hojas de estilo en cascada, el cual permite manipular el aspecto y la disposición de los elementos incluidos en archivos HTML utilizados para construir una página web. Las habilidades desarrolladas en el uso de este lenguaje se orientan a la decoracion y acomodo de las páginas web creadas.

Javascript

Es un lenguaje de programación orientado a la implementación de funciones en páginas web. Se suele utilizar a través de la implementacion de frameworks, lo que permite fluidez al programar las funciones requeridas, tales como Vue, React o Angular, etc. Mis habilidades en su uso se basan en la construccion de funcionalidades través de sus frameworks.

Angular

Framework Javascript que permite estructurar el código de manera mas limpia, eficiente y rápida para crear contenido y funcionalidades web. Mis habilidades están basadas en la administración y mejora de una plataforma web empresarial

Vue

Framework Javascript orientado a la creación de interfaces, mantiene el código simple y flexible, lo que facilita la integración en proyectos existentes. Mis habilidades en el uso del framework se orientan a la creación de páginas web con consumos de API's y plataforma web de administración empresarial.
